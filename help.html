This sample application allows to create/read/delete posts. Posts are persistent in a database. There are two possible use of the application: - interactive - through REST API using a widely available tool - curl To start the application, just run it: >/tapp It will run forever as it's a simple Web Server. Before anything can be done with the application - wait till it prints Connected to SQLITE3 DB (A) Interactive use Run in a browser (URL):8080/posts/x (URL):8080/post (URL):8080/ (URL):8080/help
